Internal class to register saving functionality. This is usually called by the 0-save-games.lua service.
SaveManager.save (slot, name)
Saves the current state of the game into a slot.

Parameters:

  • slot: The save game slot. 0 is autosave.
  • name: The human readable name of the save game. This is provided by the user.
